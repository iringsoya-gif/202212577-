# PRD — NeuralNetLab: 신경망 기초 학습 시각화 앱
**Product Requirements Document**

> 버전: 1.0 | 작성일: 2026-03-28
> 기술 스택: Python 3.12 + PySide6 + matplotlib + NumPy

---

## 1. 제품 개요

### 목적
Week 3의 5개 Python 스크립트(`01~05_*.py`)를 하나의 **인터랙티브 데스크톱 앱**으로 통합하여,
신경망 기초 개념(퍼셉트론 → 활성화 함수 → 순전파 → MLP → 보편 근사)을
슬라이더·애니메이션·실시간 시각화로 직관적으로 탐구할 수 있게 한다.

### 핵심 가치
```
"파라미터를 직접 조작하면서 신경망이 어떻게 작동하는지 눈으로 배운다"
```

### 대상 사용자
- AI/ML 수업 Week 3 수강생
- 신경망 원리를 처음 배우는 학습자

## 3. 탭별 요구사항

### Tab 1 — 퍼셉트론 (01_perceptron.py)

**핵심 기능**
- 게이트 선택 (AND / OR / XOR) 버튼
- 결정 경계 + 분류 영역 실시간 시각화
- 학습 과정 애니메이션 (QTimer, 100ms/step)
- 학습 로그 테이블 (Epoch | w1 | w2 | b | 오류수)
- XOR 선택 시 "단일 퍼셉트론으로 분리 불가" 경고 + 실패 시각화

**인터랙션**
- 학습률(η) 슬라이더: 0.01 ~ 1.0
- Epoch 수 스핀박스: 10 ~ 500
- [▶ 학습] / [⏸ 일시정지] / [↺ 초기화] 버튼
- 학습 완료 후 "정확도: X%" 배지 표시

---

### Tab 2 — 활성화 함수 (02_activation_functions.py)

**핵심 기능**
- 4종 함수 동시 비교 그래프 (Sigmoid / Tanh / ReLU / Leaky ReLU)
- 도함수(Gradient) 그래프 나란히 표시
- Vanishing Gradient 데모: 입력값 범위에 따른 기울기 소멸 시각화

**인터랙션**
- x 범위 슬라이더: -10 ~ +10
- 함수별 ON/OFF 토글 체크박스
- Leaky ReLU α 슬라이더: 0.001 ~ 0.5
- 특정 x 값 입력 → 각 함수의 출력값·기울기 실시간 표시

---

### Tab 3 — 순전파 (03_forward_propagation.py)

**핵심 기능**
- 신경망 구조 다이어그램 (QPainter 직접 렌더링)
- 입력값 변경 시 각 층의 z, a 값 실시간 업데이트
- [▶ 단계별 실행] 버튼 → Layer 1 → ReLU → Layer 2 → Sigmoid 순서로 하이라이트
- 행렬 계산 수식 패널 (현재 값으로 채워진 실제 계산 표시)

**인터랙션**
- 입력 x1, x2 슬라이더: -2.0 ~ 2.0
- 가중치 W1, W2 랜덤 초기화 버튼
- 네트워크 구조 선택: 2→3→1 / 2→4→2→1 / 2→8→4→1
- 활성화 함수 선택: ReLU / Tanh / Sigmoid

---

### Tab 4 — MLP + 역전파 (04_mlp_numpy.py)

**핵심 기능**
- XOR 학습 애니메이션 (Loss 실시간 감소 곡선)
- 결정 경계 컬러맵 실시간 업데이트 (학습 진행에 따라)
- 역전파 수식 패널 (현재 δ, dW 값 표시)
- 은닉층 뉴런별 활성화 히트맵

**인터랙션**
- 은닉 뉴런 수 슬라이더: 2 ~ 16
- 학습률 슬라이더: 0.01 ~ 2.0
- Epoch 스핀박스: 100 ~ 20,000
- [▶ 학습] / [⏸] / [↺] 버튼
- 학습 완료 시 4개 입력에 대한 예측값 테이블

---

### Tab 5 — 보편 근사 (05_universal_approximation.py)

**핵심 기능**
- 근사 대상 함수 선택 (Sine / Step / Complex / 사용자 정의)
- 뉴런 수별 근사 곡선 비교 (3 / 10 / 50 동시 표시)
- MSE 비교 바 차트

**인터랙션**
- 뉴런 수 슬라이더: 1 ~ 100 (실시간 재학습)
- Epoch 슬라이더: 1000 ~ 20,000
- [▶ 학습] 버튼 → 진행 바 표시
- 사용자 정의 함수 텍스트 입력: `sin(x) + cos(2*x)` 등

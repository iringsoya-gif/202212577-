훅의 법칙 (Hooke's Law)

탄성 한계 내에서 용수철의 변형량은 가해진 힘에 정비례합니다. Robert Hooke이 1678년 발견한 고전역학의 기본 법칙입니다.

F = k · x
F : 힘 [N] = m × g
k : 용수철 상수 [N/m]
x : 늘어난 길이 [m]
🧮

늘어남 예측 공식

질량 m을 달았을 때 중력 F = mg가 작용합니다. 훅의 법칙에 대입하면 늘어나는 길이 x를 정확히 계산할 수 있습니다.

x = (m × g) / k
m : 질량 [kg]
g : 중력가속도 9.81 m/s²
k : 스프링 상수 50 N/m
🤖

TensorFlow 선형 회귀

단일 Dense 레이어(파라미터 2개)로 구성된 신경망이 데이터로부터 k를 자동 학습합니다. MSE 손실·Adam 옵티마이저를 사용합니다.

ŷ = w · m + b
w → g/k = 0.1962
b → 편향 ≈ 0
Loss : MSE → Adam
